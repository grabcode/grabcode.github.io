## The Vision

Style Atlas was an **award-winning fashion fitting solution** for online shoppers. We provided fashion retailers with tools to **significantly reduce returns**, greatly improve shoppers' online experience, and streamline fashion labels' design & production workflow through a revolutionary collaborative cloud solution.

## My Role & Impact

As co-founder and CTO, I led the product development team and managed a small team of software engineers. I was **hands-on the code daily** while also contributing to the company's commercial and marketing strategies. I made key decisions about the technology stack, designed real-time data synchronisation systems, and drove the product from concept to an award-winning solution.

## Tech Stack

Built on Meteor.js for rapid, real-time application development:
- **Meteor.js**: Full-stack JavaScript framework powering real-time capabilities
- **JavaScript/ES6**: Modern client and server code
- **MongoDB**: Flexible document database for fashion metadata and user data
- **Real-time Synchronisation**: Live updates across all connected clients
- **Quadrant**: Custom analytics and discovery component (first component built)

## Key Features

- **Fashion fitting engine**: Intelligent size recommendations to reduce returns
- **Retailer dashboard**: Analytics and insights into sizing patterns and return rates
- **Real-time collaboration**: Cloud-based tools for fashion design and production workflows
- **Shopper experience**: Personalised fitting recommendations for online shoppers
- **Collection management**: Streamlined design-to-production pipeline

## The Impact

Over 3+ years, Style Atlas evolved from an idea into an award-winning product that addressed one of e-commerce's biggest challenges — online returns. The experience deepened my expertise in building B2B SaaS products, managing cross-functional teams, and balancing technical innovation with commercial viability.
